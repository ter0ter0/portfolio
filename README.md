# Ramengram
 ラーメン専用のトークアプリです。<br >
 投稿・返信・リポスト・お気に入り、ブックマークなどできます。 <br >
 レスポンシブ対応しているのでスマホからもご確認いただけます。

# 使用技術
- PHP 8.2.4
- Laravel 6.20.44
- MySQL 5.7
- Adminer 4.7.5
- Apache 2.4.56
- Docker/Docker-compose
- fly.io（現在停止中）

# 機能一覧
- ユーザー登録、ログイン機能
- 投稿機能
  - コメント
  - 画像投稿
  - 動画投稿
- いいね機能
  - ランキング機能
- フォロー機能
- ページネーション機能
- リポスト機能
- ブックマーク
- ラーメン記録投稿機能
